Bootstrap

É o framework mais conhecido do mundo para criar sites responsivos e mobile.
Em seu site, afirma-se: “Bootstrap torna o desenvolvimento front-end mais rápido e fácil. Ele é feito para pessoas de todos os níveis de habilidade, dispositivos de todos os formatos e projetos de qualquer tamanho”

Como utilizar


Para utilizar-se dos estilos do Bootstrap, é necessário adicionar a seguinte linha de código: 
<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">
Alguns componententes requerem JavaScript, para isto as seguintes linhas de código deverão ser adicionadas no fim da página antes da tag body 
  <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js" integrity="sha384-q8i/X+965DzO0rT7abK41JStQIAqVgRVzpbzo5smXKp4YfRvH+8abtTE1Pi6jizo" crossorigin="anonymous"></script>

  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.3/umd/popper.min.js" integrity="sha384-ZMP7rVo3mIykV+2+9J3UJ46jBk0WLaUAdn689aCwoqbBJiSnjAK/l8WvCWPIPm49" crossorigin="anonymous"></script>

Botões
Existem vários tipos de botões, variando cor, tamanho, estado inativo/ativo e outros fatores.
Seguem alguns exemplos:
Botão normal

<button type="button" class="btn btn-primary">Primary</button>
Botão pequeno

<button type="button" class="btn btn-primary btn-sm">Botão pequeno</button>
Botão grande

<button type="button" class="btn btn-primary">Botão grande</button>
Botão normal (Perigo)

<button type="button" class="btn btn-danger">Perigo</button>
Botão outline

<button type="button" class="btn btn-outline-primary">Primary</button>
Botão Estado Ativo
Link
Este botão irá demonstrar o seu estado de ativo
<a href="#" class="btn btn-secondary btn-lg active" role="button" aria-pressed="true">Link</a>
Botão Estado Desativado

Este botão aparecerá como se estivesse desativado.
<button type="button" class="btn btn-secondary btn-lg" disabled>Botão</button>
Botão Alternar Estados

Este botão irá alternar seu Estado para ativo quando clicado.
<button type="button" class="btn btn-primary" data-toggle="button" aria-pressed="false" autocomplete="off">Unica alternância</button>
Botão Tag "a"
Link
Demonstração que não só o button pode ser um botão
<a class="btn btn-primary" href="#" role="button">Link</a>
Botão Checado

Este botão aparecerá como clicado.
<div class="btn-group-toggle" data-toggle="buttons"> <label class="btn btn-secondary active"> <input type="checkbox" checked autocomplete="off">Checado</label> </div>
Botão Checado em grupo
<div class="btn-group btn-group-toggle" data-toggle="buttons"> <label class="btn btn-secondary active"> <input type="radio" name="options" id="option1" autocomplete="off" checked> Ativo </label> <label class="btn btn-secondary"> <input type="radio" name="options" id="option1" autocomplete="off" checked> Radio </label> <label class="btn btn-secondary"> <input type="radio" name="options" id="option1" autocomplete="off" checked> Radio </label>
Botão block-level

Botão que assume o tamanho do bloco.
<button type="button" class="btn btn-primary btn-lg btn-block">Botão block-level</button> <button type="button" class="btn btn-lg btn-lg btn-block">Botão block-level</button>
